# The chart type should match the question

Line and area charts are best for trends, bars are best for category comparisons, scatter charts are best for relationships, pies are best for simple proportions, and funnels are best for staged drop-off.

![Chart type controls](images/tip-chart-settings.png)

If a chart feels noisy, it is often asking the wrong visual question. Switch type before rewriting the query; the same result set can become much easier to read.
