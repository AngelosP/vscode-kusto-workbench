# You can ask Copilot to refine KQL with schema context

Use the Copilot button inside a query section when the task is local to that query. Copilot can use the selected connection and database schema while it drafts or refines KQL.

![Copilot entry point for query help](images/tip-copilot-chat.png)

Keep prompts specific. Mention tables, columns, filters, and output shape when you know them, then ask for an explanation when you want to understand the pattern instead of only pasting the result.
