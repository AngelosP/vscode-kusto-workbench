# A notebook can stay portable in one file

Use `.kqlx`, `.sqlx`, and `.mdx` files when the work is more than a plain query. The file can hold sections, settings, charts, markdown, transformations, and optional cached results together.

![Kusto notebook with sections in one editor](images/tip-editor-scratchpad.png)

Plain `.kql`, `.csl`, and `.sql` files still open normally. Reach for the notebook formats when you want the analysis, not just the text, to travel with the file.
