# URL sections can turn CSV files into analyzable tables

Paste a CSV URL into a URL section and Kusto Workbench treats it like tabular data. You get table search, scrolling, transformations, and chart data sources over data that did not come from Kusto. It is a great bridge for incident notes, exports, and small reference tables. Load the file once, transform it, and compare it beside live query results.
