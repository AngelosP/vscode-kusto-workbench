# Prettify cleans up KQL without changing the idea

When a query grows by copy, paste, and one more `where`, the Prettify action can make it readable again. It applies Kusto-aware formatting around common operators so the flow is easier to scan.

![Prettify query action](images/tip-editor-prettify.png)

Use it before asking someone else to review a query. The logic stays yours, but the layout stops making everyone work so hard.
