# Use section Copilot for surgery and the agent for orchestration

Section Copilot is best when you are editing one query with one database schema. The Kusto Workbench agent is better when the task spans multiple sections, charts, transformations, or databases.

![Kusto Workbench agent in VS Code chat](images/tip-agent-orchestration.png)

The simple rule: ask Copilot to improve the current query; ask the agent to build or reshape the notebook.
