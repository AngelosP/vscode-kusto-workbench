# Charts can start directly from query results

Add a chart section below a query when the shape matters more than the rows. Kusto Workbench links the chart to the result above it, so you can configure the visual without copying data anywhere.

![Chart section built from query results](images/tip-chart-builder.png)

Start simple: pick the chart type, choose the X column, then add one or more Y columns. You can refine the query or the visual independently as the story becomes clearer.
