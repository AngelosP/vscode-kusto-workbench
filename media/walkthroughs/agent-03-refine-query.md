# Refine with follow-up requests

The agent can resume working on a file, so you can iterate naturally.

## Try a follow-up

Now ask the agent to refine the previous result:

> *"Change this to a rolling 7-day average and extend the time range to the past 90 days"*

The agent will update the existing query and chart, the rolling window smooths out daily noise so trends become clearer.

## What you can ask for

* **Change calculations**: rolling averages, percentiles, cumulative sums
* **Adjust time ranges**: go further back, narrow to specific periods
* **Add breakdowns**: split by region, OS, version, or any dimension
* **Switch chart types**: from line to bar, add a second series, etc.
* **Fix itself / auto-heal:** ask it to fix a mistake, or use a different technique, or look for a different set of data
