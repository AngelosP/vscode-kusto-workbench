# Add a Kusto Connection

To get started, you need to connect Kusto Workbench to a cluster and database.

## How to add a connection

1. Use **Manage Connections** from the Activity Bar, or run the command **"Kusto Workbench: Manage Connections"** from the Command Palette.
2. Use either **Add new connection** (to import a single connection) or **Import connections** to bring over your connections from Kusto Explorer.

That's it, you're connected and ready to query.
